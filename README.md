# Machine-Learning-Assignment-1

Dataset Source:
The dataset was obtained from an online source (Kaggle) and contains information about Steam games.
https://www.kaggle.com/datasets/deepann/80000-steam-games-dataset/data?select=steam_data.csv

number of rows:  81048,
number of columns:  12

Column Descriptions
1. url:
Provides the direct link to the game’s page on Steam. Mainly useful for reference.
2. name:
Represents the title of the game and is used for identification purposes.
3. categories:
Includes the main features of the game, such as Single-player or Multiplayer, helping to understand the type of gameplay offered.
4. img_url:
Contains the link to the game’s image cover.
5. user_reviews:
Indicates the general user feedback, usually summarized as Positive, Mixed, or Negative, giving an overview of player satisfaction.
6. all_reviews:
Shows the overall review summary.
7. date:
Represents the release date of the game and can be used to analyze trends over time.
8. developer:
Identifies the developer of the game, which can be useful when comparing performance across different developers.
9. publisher:
Specifies the company responsible for publishing the game.
10. price:
Shows the price of the game. May include symbols or text, so it requires cleaning before performing any numerical analysis.
11. pegi:
Represents the age rating assigned to the game (12, 16, 18), indicating the appropriate audience.
12. pegi_url:
Provides a link to additional details about the PEGI rating.

Purpose of Using the Dataset:
This dataset includes information about Steam games such as names, prices, reviews, and categories. It was chosen because it contains different types of data and some cleaning issues, making it suitable for basic analysis and practice.
